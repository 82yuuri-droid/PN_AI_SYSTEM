# A-LIGHT Growth Opportunities and KPI Recommendations

## Growth Opportunities

## 1. Turn Brand Awareness Into Recruiting Intent

A-LIGHT should not pursue awareness as a vague metric. Brand awareness should mean that the right talent can understand:

- What A-LIGHT does
- What kind of people succeed there
- What working there feels like
- Why applying is worth considering now

Opportunity:

- Create recurring content that links culture, people, roles, and application CTAs.

## 2. Make Employees and Talent Stories the Core Asset

Recruiting content is strongest when candidates can see real people.

Opportunity:

- Create interview-style short videos and carousel posts featuring employees, managers, talent, or team members.
- Turn each interview into multiple reusable assets:
  - Short video
  - Quote post
  - Day-in-the-life post
  - FAQ answer
  - Job-posting copy
  - Interview preparation material

## 3. Clarify the Candidate Persona

If the target candidate is vague, content becomes generic.

Opportunity:

- Define 2-3 candidate personas.
- Create content for each persona's fears, ambitions, questions, and decision triggers.

Example personas:

- "First serious career step" candidate
- "Creative/talent-oriented growth seeker"
- "Experienced candidate looking for a better culture fit"

## 4. Build an Application Conversion Path

SNS content alone will not increase applications if the next step is unclear.

Opportunity:

- Add clear CTAs.
- Create a simple application landing page or link hub.
- Use recurring "how to apply" content.
- Reduce uncertainty around screening, interview process, and role expectations.

## 5. Use Recruiting Proof, Not Generic Branding

Candidates need proof.

Opportunity:

- Show growth stories.
- Show team support.
- Show behind-the-scenes training or work process.
- Show common candidate concerns being answered directly.

## 6. Improve Follow-Up Speed and Candidate Experience

Applications can be lost after the SNS conversion point.

Opportunity:

- Build candidate response templates.
- Track application-to-interview conversion.
- Improve interview preparation communication.
- Use AI to summarize candidate questions and recurring objections.

## KPI Recommendations

## Executive KPIs

| KPI | Purpose | Target Direction |
| --- | --- | --- |
| Monthly talent applications | Main outcome | Increase |
| Qualified application rate | Quality control | Increase |
| Interview conversion rate | Funnel health | Increase |
| Hire conversion rate | Final business outcome | Increase |
| Cost per qualified applicant | Efficiency | Decrease |

## SNS Funnel KPIs

| Funnel Stage | KPI | Why It Matters |
| --- | --- | --- |
| Awareness | Reach / impressions | Measures candidate visibility |
| Awareness | Profile visits | Measures curiosity |
| Trust | Saves | Measures content usefulness |
| Trust | Comments / replies | Measures engagement depth |
| Trust | Video completion rate | Measures story strength |
| Intent | Link clicks | Measures application interest |
| Intent | DM inquiries | Measures candidate consideration |
| Conversion | Applications from SNS | Measures direct recruiting impact |

## Content System KPIs

| KPI | Purpose |
| --- | --- |
| Posts published per week | Consistency |
| Recruiting story assets created | Asset creation |
| Employee/talent interviews captured | Proof supply |
| Content reused across channels | Efficiency |
| Content-to-application attribution | Learning |

## Operational KPIs

| KPI | Purpose |
| --- | --- |
| Content approval time | Prevents publishing delays |
| KPI input completion rate | Keeps learning loop alive |
| Application response time | Candidate experience |
| Monthly improvement actions completed | Ensures reporting becomes action |

## Priority KPI Dashboard

Start with these 8 metrics:

1. Monthly applications
2. Qualified applications
3. Interviews booked
4. Hires or next-stage candidates
5. SNS reach
6. Profile visits
7. Link clicks / DM inquiries
8. Applications attributed to SNS
