# PN Growth Producer AI

This folder turns PN Growth Producer AI into a daily executable system.

Use this AI employee to:

- Analyze sales calls
- Draft proposals
- Create SNS strategies
- Generate content ideas
- Improve client results

## Daily Rule

PN Growth Producer AI should be used every business day for at least one revenue-related task.

Priority order:

1. Sales call analysis
2. Proposal drafting
3. Client improvement recommendations
4. SNS strategy
5. Content ideas

If there is a sales opportunity, do not use the AI employee for low-priority content ideation first.

## Core Files

- `system_prompt.md`: copy into the AI tool as the system/developer instruction.
- `user_prompt_templates.md`: copy-ready task prompts for daily use.
- `workflow_diagrams.md`: Mermaid diagrams for the core workflows.
- `required_folder_structure.md`: where inputs and outputs should live.
- `operating_instructions.md`: daily, weekly, and monthly operating rules.

## Folder Map

- `01_inbox`: raw sales notes, call transcripts, client notes, KPI exports
- `02_outputs`: AI-generated working outputs
- `03_templates`: reusable input and output templates
- `04_knowledge`: offer menu, proof, objections, client patterns
- `05_review_log`: human review notes and improvement history

## Human Review Rule

AI can draft and recommend. Humans approve final pricing, final client communication, strategic promises, scope changes, and publishing.
