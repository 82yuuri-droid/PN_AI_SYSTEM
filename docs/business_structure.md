# PN Business Structure

## Mission

良いモノ・人・文化を埋没させない。

PNは、価値ある商品・人材・文化的活動が市場やSNS上で見つからない状態をなくすために、発信、採用、動画、AI活用を支援する。

## 事業領域

### 1. SNS運用

主な価値:

- クライアントの商品、サービス、人、文化を継続的に認知させる
- 投稿、ショート動画、キャンペーン、分析改善を通じて売上や応募につなげる

主な業務:

- アカウント戦略設計
- 投稿企画
- 台本作成
- 画像・動画制作
- 投稿スケジュール管理
- コメント、DM、反応の確認
- レポート作成
- 改善提案

利益への影響:

- 継続契約化しやすく、LTVを伸ばしやすい
- 運用品質が属人化すると利益率が下がる
- レポートと改善提案を自動化できると継続率が上がる

### 2. 採用支援

主な価値:

- 採用したい人に、企業や仕事の魅力を伝える
- 応募数だけでなく、採用決定や定着に近い候補者接点を増やす

主な業務:

- 採用ターゲット設計
- 求人訴求作成
- 採用SNS、採用動画、求人媒体の改善
- 応募者導線の改善
- 候補者対応テンプレート整備
- 採用成果のレポート

利益への影響:

- 成果が見えやすいと単価を上げやすい
- 応募後の対応品質まで整えると採用成果が上がる
- 採用広報コンテンツをSNS運用と横展開できる

### 3. 動画制作

主な価値:

- SNS、採用、営業、教育に使える動画資産を作る
- 企画から編集までの速度と品質で差別化する

主な業務:

- ヒアリング
- 企画構成
- 台本作成
- 撮影準備
- 編集
- サムネイル、タイトル、投稿文作成
- 修正対応
- 納品管理

利益への影響:

- 修正回数と編集工数が利益率を大きく左右する
- 台本、構成、チェックリストを標準化すると品質が安定する
- SNS運用や採用支援に動画を組み込むと客単価が上がる

### 4. AI活用支援

主な価値:

- クライアントや社内業務の工数をAIで削減する
- 属人業務を手順化、自動化し、再現性を高める

主な業務:

- 業務ヒアリング
- AI活用余地の診断
- プロンプト、テンプレート、ツール設計
- 自動化ワークフロー実装
- 社内教育
- 効果測定

利益への影響:

- 高単価化しやすい
- PN自身の業務改善で利益率に直接効く
- 他事業の制作効率を底上げできる

## KPI Tree

### 売上

売上 = 新規受注数 x 平均単価 + 継続売上

改善レバー:

- 提案資料の品質向上
- 営業リストと提案先の精度向上
- SNS、採用、動画、AI活用支援のパッケージ化
- 成果レポートによる継続提案
- 既存顧客への追加提案

### 利益

利益 = 売上 - 制作工数 - 外注費 - 管理工数

改善レバー:

- 企画、台本、レポート、修正対応の自動化
- 制作テンプレート化
- 案件別の工数、粗利の可視化
- 修正回数の削減
- 高粗利サービスへの誘導

### 初回継続率

初回継続率 = 初回契約後に継続契約へ進んだ顧客数 / 初回契約顧客数

改善レバー:

- 初月のオンボーディング品質
- 初月成果の見える化
- 2か月目以降の改善提案
- 進行状況の透明化
- クライアント確認待ちの滞留削減

### 採用成果

採用成果 = 応募数、面談数、採用決定数、採用単価、候補者品質

改善レバー:

- ターゲット別訴求の改善
- 求人、SNS、動画の一貫性
- 応募導線の改善
- 候補者対応の速度向上
- 採用レポートの定期化

### 制作効率

制作効率 = 納品物数 / 投入工数

改善レバー:

- テンプレート化
- AI下書き
- 自動チェック
- タスク管理
- ナレッジ再利用

## 初期仮説

PNで最も先に整えるべき領域は、全事業に共通する「案件管理、制作進行、レポート、ナレッジ再利用」です。

理由:

- 売上、利益、継続率、制作効率のすべてに効く
- SNS運用、採用支援、動画制作、AI活用支援で横断利用できる
- 人員を増やさずに案件数を増やす土台になる
