# Automation Backlog

優先度は、売上向上、利益向上、工数削減、品質向上、再現性向上の順で判断する。

## P0: まず作るべき業務基盤

### 1. 案件台帳

対象業務:

- 全事業共通

自動化内容:

- 顧客名、案件種別、契約金額、原価、粗利、担当、開始日、納期、ステータスを一元管理
- 案件ごとの工数、納品物、修正回数を記録
- 月次売上、粗利、継続率を自動集計

期待効果:

- 利益の見える化
- 赤字案件の早期発見
- 高粗利メニューの把握

最初の実装:

- CSVまたはスプレッドシート形式の案件台帳テンプレート
- 月次集計スクリプト
- 粗利率アラート

### 2. 制作進行管理

対象業務:

- SNS運用
- 採用支援
- 動画制作

自動化内容:

- 企画、台本、制作、確認、修正、納品のステータス管理
- 締切前リマインド
- クライアント確認待ちの検知
- 修正回数の記録

期待効果:

- 納期遅延の削減
- 管理工数の削減
- 修正対応の利益圧迫を可視化

最初の実装:

- タスク台帳テンプレート
- 納期アラート
- 停滞タスク一覧

### 3. 月次レポート自動生成

対象業務:

- SNS運用
- 採用支援
- AI活用支援

自動化内容:

- 数値入力からレポート骨子を自動生成
- 成果、課題、次月施策をテンプレート化
- 継続提案の文章を自動生成

期待効果:

- 初回継続率の向上
- レポート作成工数の削減
- 改善提案の品質安定

最初の実装:

- KPI入力CSV
- Markdownレポート生成
- 提案文テンプレート

## P1: 収益を伸ばす自動化

### 4. 提案資料ドラフト生成

対象業務:

- 営業
- ChatGPT連携
- SNS運用
- 採用支援
- AI活用支援

自動化内容:

- ヒアリングメモから提案骨子を生成
- 事業別パッケージを自動選定
- 見積もり案と期待成果を出す

期待効果:

- 提案速度向上
- 受注率向上
- 客単価向上

### 5. 既存顧客への追加提案検知

対象業務:

- 全事業共通

自動化内容:

- レポート結果や案件状況から追加提案候補を抽出
- SNS運用から動画制作、採用支援から採用動画などの横展開を提案

期待効果:

- 既存顧客売上の増加
- LTV向上

### 6. 営業リスト評価

対象業務:

- 営業

自動化内容:

- 見込み顧客を業種、SNS状況、採用状況、動画活用状況でスコアリング
- 提案テーマを自動生成

期待効果:

- 営業工数削減
- 商談化率向上

## P2: 制作品質を安定させる自動化

### 7. SNS投稿企画生成

対象業務:

- SNS運用

自動化内容:

- 顧客情報、ターゲット、過去投稿から投稿案を生成
- 投稿カレンダー化
- 投稿文、ハッシュタグ、ショート動画台本を生成

期待効果:

- 企画工数削減
- 投稿品質の安定

### 8. 採用訴求テンプレート

対象業務:

- 採用支援

自動化内容:

- 職種、ターゲット、企業の強みから求人訴求を生成
- ペルソナ別の訴求比較
- 採用SNS投稿案を生成

期待効果:

- 採用成果向上
- 提案品質の標準化

### 9. 動画制作チェックリスト

対象業務:

- 動画制作

自動化内容:

- 企画、撮影、編集、納品前チェックをテンプレート化
- 修正理由を分類
- 修正が多い工程を可視化

期待効果:

- 修正回数削減
- 品質向上
- 利益率改善

## P3: AI活用支援の商品化

### 10. 業務AI診断シート

対象業務:

- AI活用支援

自動化内容:

- クライアント業務をヒアリングし、自動化余地をスコアリング
- 優先順位付き改善提案を生成
- 導入ロードマップを作成

期待効果:

- AI活用支援の受注率向上
- 高単価化
- PN自身の業務改善にも転用可能

## 優先実装順

1. 案件台帳
2. 制作進行管理
3. 月次レポート自動生成
4. 提案資料ドラフト生成
5. 業務AI診断シート
